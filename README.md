O projeto a seguir consiste nessas etapas:

1. Entrada (inputs)
2. Reconhecimento das informações ja pré-definidas em Class
3. Criação de um objeto baseada nas informações do usuário
4. Uso de ".push" para adicionar essas informações ao array "banco"
5. Salvamento dos dados dentro de array no Local Storage do navegador para manter as informações
mesmo após atualizar a página ou fechar o navegador.
